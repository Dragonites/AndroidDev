Een werkende versie van het examen. Stappenplan_Elections2016_hard.txt
bevat de volgorde dat de bestanden zijn aangemaakt. Dit is ook
de volgorde waarin je de bestanden moet openen.
Alle bestanden zijn voorzien van commentaar om de werking duidelijker te maken.

Dit is de 'hard'-versie en omvat:
- Custom opstarticoon
- Fragments
- Intents (Opstarten van nieuwe Activities)
- Dynamische UI (Master-Detail
		portrait = Activity A -> Fragment A, Activity B -> Fragment B
		landscape = Activity A -> Fragment A + Fragment B)
- SharedPreferences (Opslaan van settings en terug inladen)
// Dit is nieuw in 'hard'-versie
- Gebruik van aangelevede bestanden
- SQLite
- ContentProvider (Contract, DBHelper, Provider)
